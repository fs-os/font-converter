# Font converter
Converts a `(y: 8, x: 7)` font to a `(y: 10, x: 8)`.

### Building
```console
$ git clone https://github.com/fs-os/font-converter
$ cd font-converter
$ make
...

$ make run
"\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00"  /* NULL */
"\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00"  /* NULL */
"\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00"  /* NULL */
...
```
